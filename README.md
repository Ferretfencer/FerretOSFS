# FerretOSFS (Operating System For Slabs)
The next step in privacy based mobile operating systems
This OS will be designed to be fully open source so all code can be vetted for either vulnerabilities or, if trust is low, making sure there are no hidden trackers or other potentially malicious code
While it will be designed for a specific line of devices, it should also be able to be installed on devices that run Android and can have an unlocked bootloader, similar to how Graphene works, though it may have issues on there
Feel free to share any help with its production, as it will be its own open source project, separate from Google's AOSP
